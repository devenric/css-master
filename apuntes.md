# random 
propiedad:random (valor1, valor2)
para crear saltos entre randomizaciones (ejemplo: de 50px en 50px) se usa **by**

Ejemplo: 

*size: random( 100px, 300px, by 50px)*

Esto significa que puede randomizar desde 100 px hasta 300px 
(100, 150, 200, 250, 300)
